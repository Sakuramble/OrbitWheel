# OrbitWheel 1.1.2

## 新增文件夹目标

- 动作类型新增“打开文件夹”
- 在设置中选择该类型后会直接弹出文件夹选择窗口
- 执行时使用资源管理器打开对应文件夹
- 圆环内优先显示系统文件夹图标，无法读取时使用内置文件夹图标兜底

## 圆环边缘自适应

- 当鼠标靠近屏幕边缘，导致圆环无法完整显示时，OrbitWheel 会自动修正圆心位置
- 修正时会同步移动鼠标和圆环，确保圆心仍在鼠标处
- 圆环会移动到刚好可完整显示的位置，避免扇区被屏幕边缘裁切

## 继续保留

- 通用托盘双击唤醒：已运行但折叠到托盘的软件会通过 Windows 托盘项模拟双击恢复
- 软件已显示时直接切换窗口，不触发托盘操作
- Store/AppsFolder 应用真实进程名解析
- 数字键 `1–6` 直接执行扇区
- 鼠标滚轮和左右方向键翻页
- 全新深色玻璃 UI 与系统操作图标
