# OrbitWheel 1.0.2

## 微信与 QQ 托盘唤醒修复

- 微信运行时通过 Windows 托盘项唤醒，由微信自身恢复正确主窗口，避免灰色辅助页面
- QQ 运行时仅恢复系统认定的主窗口，不再选择不可关闭或不可最小化的内部窗口
- 兼容自动隐藏任务栏与高 DPI 缩放，托盘操作完成后恢复鼠标位置
- 微信与 QQ 已运行时不再启动新的应用实例

## 本机验证

- 微信：关闭至托盘后成功恢复正常“微信”主窗口
- QQ：主窗口最小化后成功恢复，窗口仍可正常最小化与关闭
- 微信与 QQ 执行前后进程数量均保持不变

本版本包含 OrbitWheel 1.0.1 的桌面程序与 Store/UWP 应用运行检测。
