# OrbitWheel 1.0.2

## 微信与 QQ 托盘唤醒修复

- 不再只检查可见窗口
- 支持从微信的多个 `Weixin.exe` 进程中识别隐藏的 `Qt51514QWindowIcon` 主窗口
- 支持从 QQ 的多个 `QQ.exe` 进程中识别隐藏的 `Chrome_WidgetWin_1` 主窗口
- 排除托盘消息窗口、系统消息窗口、输入法窗口和过小的辅助窗口
- 找到隐藏主窗口后恢复、置顶并切换，而不是再次打开新界面

## 本机验证

- 微信：执行前后进程数量保持 `5 → 5`，隐藏“微信”主窗口成功恢复
- QQ：执行前后进程数量保持 `9 → 9`，隐藏“QQ”主窗口成功恢复

本版本包含 OrbitWheel 1.0.1 的桌面程序与 Store/UWP 应用运行检测。
