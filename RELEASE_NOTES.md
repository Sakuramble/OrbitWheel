# OrbitWheel 1.0.2

## 通用托盘应用唤醒

- 已运行但关闭到托盘的软件会优先匹配 Windows 托盘项，并执行系统单击操作
- 使用托盘辅助功能名称匹配，不依赖图标图像；QQ 有通知并更换图标时仍可识别
- 兼容自动隐藏任务栏与高 DPI 缩放
- 唤出自动隐藏任务栏期间锁定鼠标移动，操作完成或失败后均恢复鼠标位置并解除锁定
- 软件已显示时直接切换窗口，不触发托盘操作
- Store/AppsFolder 应用会从 AppID 与动作名称解析真实进程名，修复 ChatGPT 等应用被误判为未运行的问题

## 圆环键盘操作

- 从右上扇区开始按顺时针显示 `1–6`，数字位于各扇区靠圆心位置
- 主键盘与小键盘 `1–6` 均可直接执行对应扇区
- 左右方向键可向前或向后切换页面

## 本机验证

- 微信、QQ、ChatGPT、FlClash 关闭至托盘后均成功恢复正常主窗口
- QQ 在通知状态下的托盘名称仍可准确匹配
- 三个软件执行前后进程数量均保持不变
- 自动隐藏任务栏开启时，托盘恢复后鼠标物理坐标保持不变，鼠标锁定已解除

本版本包含 OrbitWheel 1.0.1 的桌面程序与 Store/UWP 应用运行检测。
